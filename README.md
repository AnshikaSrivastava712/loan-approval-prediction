# Loan Approval Prediction

A machine learning project that predicts whether a loan application will be approved or rejected based on applicant details such as income, education, credit history, and loan amount.

## Project Overview
The goal of this project is to analyze financial and personal applicant data and build classification models to predict loan approval status. The project also includes data preprocessing, visualization, feature scaling, and performance evaluation.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes

## Project Workflow

2. Data Cleaning
3. Handling Missing Values

5. Data Visualization
6. Feature Scaling
7. Model Training
8. Model Evaluation
